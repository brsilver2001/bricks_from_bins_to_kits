Welcome to the Ben's Bricks BINS TO KITS App

This page, and all the functionality behind it, was created as part of a capstone project for the Galvanize Data Science Immersion course. This includes programming and training of a TensorFlow convolutional artificial neural network for recognizing Lego ID numbers from photos, and all creation, hosting, and functionality of the website.

All work was done by Brian Silverstein starting September 2017.
